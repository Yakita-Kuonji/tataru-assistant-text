本功能需綁定付款資訊，但免費方案不會扣款 查看付費規則

使用此功能前需啟用Google Cloud Platform功能，請至 Google Colud Platform 啟用

  1. 至 API keys頁面 按照以下步驟建立一個API key
  2. 初次使用會顯示注意事項，選第一項按Continue即可
  3. 選擇Get API key
  4. 選擇Create API key
  5. 隱私條款，選擇Got it
  6. 如果Google Cloud裡沒有專案，請點選上方的【Create API key in new project】。
    若已有專案可從下方選擇已建立的專案，然後點選下方的【Create API key in existing project】繼續。
  7. API key建立完成後，點選Copy複製
  8. 複製之後貼到【Tataru Assistant】裡的【API設定】>【Gemini設定】的API KEY欄位裡(按下Ctrl+V可貼上)， 設定完畢後請記得儲存您的設定
  9. 儲存完畢後請回到【翻譯設定】將【翻譯引擎】更改為【Gemini】即可開始使用
